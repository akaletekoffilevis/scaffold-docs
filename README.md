# Scaffolder — Documentation

Site de documentation pour [Scaffolder-CLI](https://github.com/akaletekoffilevis/Scaffolder-CLI), un CLI universel pour générer des projets dans tous les langages.

## Stack

- React 19 + TypeScript
- Vite 8
- Déploiement : Vercel

## Développement

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
# sortie dans dist/
```

## Déploiement

```bash
npx vercel --prod
```

## Logos

6 variantes disponibles dans `public/` :

| Fichier | Usage |
|---------|-------|
| `logo.svg` | Hero du site (dark, bleu-violet) |
| `logo-minimal.svg` | Favicon, nav, mode sombre |
| `logo-light.svg` | Mode clair |
| `logo-neon.svg` | Variante neon |
| `logo-horizontal.svg` | README GitHub |
